# Crabtree Montessori Website

One-page marketing site for Crabtree Montessori built with React, TypeScript, Tailwind CSS, and Vite.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Notes

- The site includes a floating theme palette switcher with 9 editable palette options.
- The selected palette is saved in `localStorage` and persists after refresh.
- Hero image and video areas are implemented as polished placeholders for future assets.
